This plugin is deprecated and will no longer be maintained. Please use the [Login by Auth0](https://wordpress.org/plugins/auth0/) plugin to integrate Auth0 with your WordPress install. If you already have this plugin installed, please follow the migration steps [here](https://auth0.com/docs/cms/wordpress/troubleshoot#how-do-i-migrate-from-social-login-with-auth0-to-login-by-auth0-) to continue to receive updates.
